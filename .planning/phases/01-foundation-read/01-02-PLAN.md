**Class: CalDAVClient**

Constructor takes `CalDAVAccount` (from config.ts) and stores it as `private account`.

**Method: `async connect(): Promise<void>`**
- Load credentials from keychain via `loadCredentials(this.account.id)`
- If null, throw `new AuthError('No credentials found for account ${this.account.id}')`
- Parse credentials JSON
- If `this.account.authType === 'basic'`:
  - Call `createDAVClient({ serverUrl: this.account.serverUrl, credentials: { username: this.account.username, password: parsedCreds.password }, authMethod: 'Basic', defaultAccountType: 'caldav' })`
- If `this.account.authType === 'oauth2'`:
  - Get fresh access token via `getValidAccessToken(this.account.id)`
  - Call `createDAVClient({ serverUrl: this.account.serverUrl, credentials: { tokenUrl: parsedCreds.tokenUrl, username: this.account.username, clientId: parsedCreds.clientId, clientSecret: parsedCreds.clientSecret, refreshToken: parsedCreds.refreshToken }, authMethod: 'Oauth', defaultAccountType: 'caldav' })`
- Store the resulting `DAVClient` as `private client`
- Wrap the entire call in try/catch: catch errors and rethrow as `NetworkError` or `AuthError` depending on error message content (401/403 -> AuthError, others -> NetworkError)

**Method: `async fetchCalendars(): Promise<DAVCalendar[]>`**
- Assert client is connected (throw if `this.client` is undefined)
- Call `this.client.fetchCalendars()`
- Return result
- Wrap in try/catch with NetworkError

**Method: `async fetchCalendarObjects(calendar: DAVCalendar, timeRange?: { start: string; end: string }): Promise<DAVObject[]>`**
- Assert client connected
- Call `this.client.fetchCalendarObjects({ calendar, ...(timeRange ? { timeRange } : {}) })`
- Return result
- Wrap in try/catch with NetworkError

**Method: `async fetchSingleObject(calendar: DAVCalendar, objectUrl: string): Promise<DAVObject | null>`**
- Assert client connected
- Call `this.client.fetchCalendarObjects({ calendar, objectUrls: [objectUrl] })`
- Return first result or null
- Wrap in try/catch with NetworkError

Store the client instance so it can be reused across multiple calls. Export the class as default and named export.

1. Create `src/utils/ical-parser.ts`:

```typescript
import ICAL from 'ical.js';
import type { ParsedEvent, Attendee, EventTime } from '../types.js';
import { ParseError } from '../errors.js';
```

**Function: `parseICS(raw: string): ParsedEvent`**

Steps:
a. If `raw` is empty or not a string, throw `new ParseError('Invalid ICS input: empty or not a string')`
b. Parse: `const jcal = ICAL.parse(raw)` — wrap in try/catch, rethrow as ParseError
c. `const comp = new ICAL.Component(jcal)`
d. `const vevent = comp.getFirstSubcomponent('vevent')` — if null, throw `new ParseError('No VEVENT component found in ICS data')`
e. `const event = new ICAL.Event(vevent)`

**Timezone-preserving time extraction** (helper function `extractEventTime`):
- Get property value: `vevent.getFirstPropertyValue(propName)` as `ICAL.Time`
- If null, return null
- Get tzid: `time.timezone?.tzid ?? (time.isUtc ? 'UTC' : 'floating')`
- Get localTime: `time.toString()` — this preserves the original local time string
- CRITICAL: NEVER call `.toJSDate()` — this collapses timezone info
- Return `{ localTime, tzid }` as `EventTime`

**Attendee extraction:**
- `vevent.getAllProperties('attendee')` returns `ICAL.Property[]`
- For each property:
  - `email`: `prop.getFirstValue().replace(/^mailto:/i, '')`
  - `cn`: `prop.getParameter('cn') ?? null`
  - `role`: `prop.getParameter('role') ?? null`
  - `partstat`: `prop.getParameter('partstat') ?? null`

**Organizer extraction:**
- `vevent.getFirstProperty('organizer')`
- If present, extract same fields as attendee (email from value, cn from parameter)
- If absent, return null

**RRULE extraction:**
- `vevent.getFirstPropertyValue('rrule')?.toString() ?? null`

**Return ParsedEvent** with all fields populated. Include `raw` as the original input string.
