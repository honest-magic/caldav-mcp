Implementation approach:
- Create `VCALENDAR` component, set `VERSION:2.0`, `PRODID:-//honest-magic//caldav-mcp//EN`
- Create `VEVENT` subcomponent
- Set `UID` (params.uid or `randomUUID()`)
- Set `DTSTAMP` via `ICAL.Time.now()`
- Set `SUMMARY`
- For `DTSTART` and `DTEND`: create `ICAL.Time.fromDateTimeString(time.localTime)`. If tzid is IANA (not "UTC", not "floating"), set TZID parameter on the property. If tzid is "UTC", set `time.isUtc = true`. If "floating", leave as-is.
- Set `DESCRIPTION` and `LOCATION` only if non-null
- VTIMEZONE: Skip for now per RESEARCH.md open question #1 — document the limitation with a code comment. Most target providers (iCloud, Google) accept events without VTIMEZONE.
- Return `vcalendar.toString()`
